# 🚀 Dashboard Simple - Proyecto Web Sin React

Un dashboard básico estilo **Link Tree** creado únicamente con **HTML, CSS y JavaScript puro**, demostrando que se pueden crear aplicaciones web modernas y atractivas sin necesidad de frameworks.

## ✨ Características

### 🎯 **Página Principal (Dashboard)**
- **Diseño estilo Link Tree** con perfil de usuario
- **4 botones principales** que llevan a diferentes secciones
- **Animaciones CSS** y efectos hover interactivos
- **Diseño responsive** para todos los dispositivos
- **Gradientes modernos** y efectos de glassmorphism

### 📱 **4 Secciones con Estilos Únicos**

#### 🖥️ **Sección 1 - Desarrollo**
- **Estilo**: Terminal / Matrix / Cyberpunk
- **Colores**: Verde neón sobre fondo negro
- **Efectos**: Lluvia de caracteres Matrix, resplandor terminal
- **Tema**: Desarrollo web y programación

#### 🎨 **Sección 2 - Diseño**
- **Estilo**: Elegante / Minimalista / Revista
- **Colores**: Blanco, grises y acentos de color
- **Efectos**: Grid de contenido, sombras sutiles
- **Tema**: Principios del diseño y tipografía

#### 🌈 **Sección 3 - Creatividad**
- **Estilo**: Artístico / Creativo / Galería
- **Colores**: Gradientes vibrantes y multicolor
- **Efectos**: Formas flotantes, animaciones de color
- **Tema**: Inspiración artística y proceso creativo

#### 🔬 **Sección 4 - Tecnología**
- **Estilo**: Futurista / Sci-Fi / Tecnológico
- **Colores**: Cian neón sobre fondo negro
- **Efectos**: Grid en movimiento, partículas flotantes
- **Tema**: Tecnologías emergentes e innovación

## 🚀 Cómo Usar

1. **Clona o descarga** el proyecto
2. **Abre `index.html`** en tu navegador
3. **Explora** las diferentes secciones
4. **Interactúa** con los elementos del dashboard

### **Navegación**
- **Click** en los botones para ir a las secciones
- **ESC** para volver al dashboard desde cualquier sección
- **Scroll** para ver efectos de parallax
- **Hover** para efectos interactivos

## 🔧 Personalización

### **Cambiar Colores**
Edita las variables CSS en `styles.css`:
```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --accent-color: #00ff00;
}
```

### **Agregar Nuevas Secciones**
1. Crea un nuevo archivo HTML
2. Copia la estructura de una sección existente
3. Personaliza estilos y contenido
4. Agrega el enlace en `index.html`

### **Modificar Animaciones**
Edita los keyframes en `styles.css`:
```css
@keyframes tuAnimacion {
  0% { /* estado inicial */ }
  100% { /* estado final */ }
}
```
